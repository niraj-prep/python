# Python College Practicals & Assignments

Welcome to the **Python College Practicals & Assignments** repository! This repository contains a comprehensive collection of Python scripts and programs developed as part of my college curriculum. It includes various lab practicals, assignments, and mini-projects covering fundamental to advanced Object-Oriented Programming (OOP) concepts, Data Structures, and Data Analysis using Python.

## Repository Structure

The repository is organized into distinct logical directories to easily locate assignments and practical lab codes:

### 1. `Assignment/`
This directory contains homework and class assignments that involve problem-solving and logic building.
*   **Data Analysis & Pandas**: `book-info-csv.py`, `diamond-dataframe.py`, `population-csv.py`, `cosmetic-company-sales.py`
*   **Object-Oriented Programming (OOP)**: `emp-manager-inheritance.py`, `recruitment-company.py`, `lib-management-sy.py`
*   **Math & Matrices**: `matrix3x3.py`, `matrix4x4.py`, `matrix-operation.py`, `calc.py`, `ohm's-law.py`
*   **String & Pattern Tasks**: `pattern-task1.py` to `pattern-task6.py`, `string-capitalized.py`, `string-statistics.py`
*   **File Handling**: `text-file.py`, `script-without-#.py`

### 2. `Practical/`
This directory contains hands-on lab practicals demonstrating core Python features and libraries.
*   **Core Fundamentals**: `loops.py`, `function.py`, `largest-among3.py`, `natural-no-sum.py`
*   **OOP Concepts**: `marksheet-oops.py`
*   **Data Handling & Visualization**: `exam-data-analysis.py`, `pandas-csv.py`, `line-graph.py`, `student-marksheet.py`
*   **Arrays & Strings**: `array3x3.py`, `string-operation.py`
*   **File I/O Systems**: `file-handling.py`, `bank-token-system.py`

## Getting Started

### Prerequisites
To run these scripts, you will need to have Python installed on your system.
*   **Python 3.x** (v3.8 or higher recommended)
*   **Pip** (Python package installer)

Some scripts require external libraries like `pandas` and `matplotlib`. You can install the required dependencies using pip:
```bash
pip install pandas matplotlib numpy
```

### Installation & Execution
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/niraj-prep/python.git
   ```
2. Navigate to the project directory:
   ```bash
   cd python
   ```
3. Run any Python script using the terminal or command prompt. For example:
   ```bash
   python Practical/exam-data-analysis.py
   ```
